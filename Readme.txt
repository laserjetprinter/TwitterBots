	 A variety of twitter bots. To successfully tweet from the bot, your own personal access keys and 
tokens will need to be inserted in place of the "'xxx'".

RandomWikiArticle

This bot will tweet a random phrase from an array list and pull a
random article from wikipedia -- and tweet the combo. 

/**************************************************************/

RandomRate

This bot will generate a random phrase and rating for a targeted
twitter user's tweets. The targeted user id is used to track
every time they update their account. The bot will continue to run
until the system it is on is shutdown or in sleep mode. 

/**************************************************************/

RandomLinkedIn

This bot is currently a work in progress, with the goal of promoting
a user's LinkedIn account.
